This directory contains the data files for conj.py.

  conj.csv         Conjugation names and id numbers.
  conjo.csv        Conjugation data.
  conjo_notes.csv  Conjugation to notes mapping. 
  conotes.csv      Conjugation note texts.
  kwpos.csv        Part-of-speech keywords.

Assuming this README file is not out of date, the files here are
exact copies of files of the same names in the JMdictDB project
as of revision hg-20180525-61238f of that project.
